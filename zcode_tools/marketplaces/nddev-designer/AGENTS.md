# ZCode environment — nddev-designer

This `~/.zcode` directory is produced by **nddev-zcode-app** from the
`nddev-designer` marketplace — a designer-oriented setup (placeholder scaffold,
to be filled in).

## What lives here

- `AGENTS.md` — this file. User-scope default instructions for every ZCode workspace.
- `skills/`, `commands/`, `agents/` — user-scope components (currently empty).
- `marketplace.json` — the `nddev-designer` plugin marketplace.
- `plugins/` — plugin bundles (currently empty).
- `cli/config.json` — plugin enable/disable state, hooks, MCP servers.
- `v2/` — desktop-app state (provider definitions, preferences, credentials).

## Operating rules

- English for code, docs, and commits (Conventional Commits). Sole author: Danil Silantyev.
- To change this environment, edit the source in the `nddev-designer` marketplace of
  the nddev-zcode-app repo and re-run the installer — do not hand-edit rendered files.
