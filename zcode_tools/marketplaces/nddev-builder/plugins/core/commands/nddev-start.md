Onboard into nddev-builder from zero.

Follow the `getting-started` skill exactly: read `nddev-builder-orientation` for
the flatten-to-user-scope model, then take the path that fits — add one component
with an `add-*` skill, compose a whole plugin with `scaffold-plugin`, or start a
new setup with `add-marketplace` — and always finish with `validate-components`
and `devtest-plugin`.
